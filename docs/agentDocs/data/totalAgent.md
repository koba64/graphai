# totalAgent

## Description

Returns the sum of input values

## Schema

#### inputs

```json

{
  "type": "object",
  "properties": {
    "array": {
      "type": "array",
      "description": "the array"
    }
  },
  "required": [
    "array"
  ]
}

````

#### output

```json

{
  "type": "object"
}

````
